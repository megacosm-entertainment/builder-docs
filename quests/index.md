# Quests

Quests in Sentience are built with the **QEdit** (Quest Editor). They support multi-stage progression, multiple objective types, branching stage flow, variables, scripting hooks, and configurable rewards.

Quests are JSON-driven and loaded at startup. Each quest belongs to an area and has a widevnum reference (e.g. `5#1000`).
